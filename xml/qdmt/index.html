QDMT: Quick and Dirty Markup Tokenizer

This is the code for the QDMT. It is presented in literate programming style and was processed using xmlp alpha. I think this makes the whole thing more readable, understandable and maintainable.

Eventually I'll want to "port" (basically rewrite) this in C. For now, I need it for a Perl project.
