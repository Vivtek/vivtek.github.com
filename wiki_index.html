Recent updates and today's traffic (fossil page)
This page is a fossil left over from 2008, when for a brief time I enabled modifications online using a static wiki builder of my own weird design.

The spam levels were rapidly untenable and I was distracted by the need to fix up my 140-year-old-house, so I disabled that in 2009 or 2010, and when I later moved my static page serving to Github, none of it worked anyway.

Today, 2014-04-06, I have gotten my static site builder to the point that it can build a recent-changes page, and since there's no reason to call this a Wiki any more (because it ain't), this page is now over here at Recent changes.
