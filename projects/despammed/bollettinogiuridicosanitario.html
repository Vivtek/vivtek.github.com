Ooooh! Paydirt! Obfuscated code is always fun! But it was tricky... Where is that "getDecText1" defined? By a process of elimination, it could only be in that iframe at the top -- but curling to that only got me a 500 Server Error response, even if I set a better agent and referrer in the call. I'm still not sure why.

So I gave up.

But then I realized it wasn't going to let me rest. So I Googled getDecText1, and got exactly one hit, dated last month and describing a similar malware. Hmm. There, the definition was in a file named "rijndael.js" which 404s now -- so I still had no definition for the function, but at least I know that "Dec" stands for "decrypted".

Let me reiterate that. This is not obfuscated Javascript -- it is encrypted Javascript, which wins on coolness... except... the key and decryption algorithm are, you know, published on the Internet, because otherwise the target browser cannot decrypt them.

So that was a dead end. So ... I gave up.
