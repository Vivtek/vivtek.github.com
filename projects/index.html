Programming projects
This page has been a placeholder since at least 2009. I'm still no closer to summarizing the big and little projects I've written about.

Some bigger projects I've worked on include the wftk open-source workflow toolkit, the Despammed spam filter, the LPML literate programming processor, and more. Since about 2004, I've been a full-time technical translator and no longer program for pay - although if you want to propose something, I'm listening.

Recent work includes a bunch of stuff on Github, stuff on CPAN, and a neat little utility for TRADOS I've written up here on the site. I earnestly hope there will be more. I suddenly feel like I want to start programming more again.
