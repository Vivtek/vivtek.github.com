Topic: xqmf -- Where from here?

[ project home ] [ discussion ]
Well, as of 30 July 2000, now that I have actually completed part of this project, here is what I want to do, in rough order of how I want to go about it:
  • Autoreporting of project overview.
  • Integration with CVS for version control of result files.
  • Implementation of a parser or two (Perl and C).
  • Implementation of some real metrics (complexity measures of some sort.)
  • Integration with CVS for project file retrieval.
  • Test on some nice existing open-source projects (PostgreSQL, for instance.)
  • Figure out what to do about relationships between entities (this will come naturally).
  • Work out the details of getting analyzers to and from a remote repository.
  • Do a more-or-less thorough survey of the metric field and select a longer list of metrics which should be implemented.
  • Encourage people to implement them...
What? Timeline? You still think I have any idea about time? Boy, are you naive.
