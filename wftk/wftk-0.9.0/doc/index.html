wftk: Open-source workflow toolkit

The wftk is simultaneously a workflow application and a toolkit which can be used to build specialized workflow components. The current version is v0.5, as of October 2000. You can find out more about the following topics:

  • Installation guide
  • Configuration/integration guide
  • User manual
  • Short tutorial on workflow and how it works
  • Design and implementation (including interoperability)
  • Licensing terms
  • Project home page at Vivtek
  • Download page at Vivtek
As of this writing (October 7, 2000) most of this is incomplete. The same distribution will be updated as documentation changes, so check back at the download page for information on updates. Feel free to contribute any documentation you want; experiences in installing the wftk in strange and eerie environments would be extremely welcome.
