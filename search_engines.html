Where did my Google go?

Topic: Search engines

[ additional information ] [ links ]
In brief, a search engine is a searchable database online which allows you to enter a phrase and which returns to you a ranked list of URLs which are related to it. All of this is very straightforward except for the part about "related to it." What a search engine considers related is generally not very related to a human being.

To make this work, the following components are necessary:

  • A program which finds URLs, downloads pages, and gets links to other pages out of them. This kind of program is called a spider or a robot because it automatically walks over the Web to find more pages.
  • A program which takes all the pages found and extracts words or phrases which are in them, placing all that information into a database. (See content-based search as soon as that topic is written.)
  • The Web form interface which takes a user's query, searches the database, and formats the list of links.
  • (Usually) Some way for people to submit URLs. This is useful for good coverage because not all pages are linked from other pages.

The database must somehow relate words and phrases to URLs which bear on them, and each of the search engines does this in a different way. That's why the lists of URLs which the various search engines return for a single phrase are usually very different, and why meta-engines give better results than a single engine (a meta-engine simply runs the same search for you on several search engines, then collates the results and presents them to you.)
