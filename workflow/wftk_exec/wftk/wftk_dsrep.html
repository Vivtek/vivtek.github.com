Dealing with datasheet repositories

[ Previous: Writing HTML descriptions of processes, tasks, and so on ] [ Top:  ] [ Next: Dealing with procdef repositories ]
A couple of silly little functions without which the whole thing would be useless: how to load and save datasheets. These functions are really very simple wrappers around the adaptor.
  
XML * wftk_datasheet_load (char * dsrep, char * datasheet_id)
{
   WFTK_ADAPTOR * ad;

   ad = wftk_get_adaptor (DSREP, dsrep);
   if (!ad) return NULL;

   return (wftk_call_adaptor (ad, "load", datasheet_id));
}
void wftk_datasheet_save (XML * datasheet)
{
   WFTK_ADAPTOR * ad;
   ad = wftk_get_adaptor (DSREP, xml_attrval (datasheet, "repository"));
   if (!ad) return;

   wftk_call_adaptor (ad, "save", datasheet);
}
