Language codes used by TRADOS 2007
As far as I know, until now there hasn't been a complete list of the language codes used by TRADOS 2007, so I spent a couple of hours this week with the Registry editor and discovered them all.

This is actually pretty straightforward, although tedious: set one language active at a time, start Workbench, click File|New to see the languages it will offer you for a new TM, and look it up in the list of ISO 639-1 language codes at Wikipedia.

Anyway, here is the complete list, sorted by internal TRADOS numeric code (i.e. the code used in the Registry entry), by the two-letter ISO language code for those languages that have them, and by the name of the language.
